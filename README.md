# EmployeesSampleApp
მონაცემთა ბაზის შექმნის სკრიპტი არის საქაღალდეში Scripts. პროგრამის პირველი გაშვებისას ხდება ბაზის არსებობის შემოწმება, თუ ბაზა არაა შექმნილი, მაშინ ხდება მისი შექმნა. 
ბაზაში არის 2 ცხრილი: Employees (შეიცავს ინფორმაციას თანამშრომლების შესახებ და არის დამატებული 50 ჩანაწერი) და Ranks(დამატებულია 3 როლი).
სკრიპტი ეშვება Microsoft SQL Server 2014 ან უფრო ახალ ვერსზიაზე, მაგრამ სასურველია 2019 წლის ვერსიის გამოყენება (ბაზის შესაქმნელად გამოიყენება 2019 წლის მონაცემთა ბაზის მისამართი).
მთავარ გვერდზე სახელის, გვარის და როლის ფილტრები რეაგირებენ ყოველ ცვლილებაზე, ხოლო ხელფასის დიაპაზონის ცვოილებისას აუცილებელია Enter-ის დაჭერა.
ასევე, თუ გვერდზე ასახული ჩანაწერების რაოდენობა იცვლება ხელით (და არა ამ ველში არსებული ისრების მეშვეობით), მაშინაც უნდა დააჭიროთ Enter-ს.
